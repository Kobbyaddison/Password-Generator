# Password Generator

This is a simple and secure random password generator built with HTML, CSS, and JavaScript. It allows users to generate strong passwords with customizable lengths and provides a click-to-copy feature for convenience.

## Features

* Generate two passwords at a time, giving users options to choose from.
* Adjustable password length between 4 and 50 characters.
* Includes uppercase letters, lowercase letters, numbers, and symbols.
* Click-to-copy functionality for quick copying of passwords.
* Responsive design that works across modern browsers.

## Usage

1. Open the project in a browser.
2. Enter the desired password length in the input box.
3. Click the "Generate Passwords" button.
4. Click any of the generated passwords to copy it to the clipboard.

## Technologies

* HTML – page structure
* CSS – styling and hover effects
* JavaScript – password generation and clipboard functionality

## How it Works

* The user inputs the desired password length.
* When the generate button is clicked, JavaScript selects random characters from a predefined set including letters, numbers, and symbols.
* Two passwords are displayed.
* Clicking a password copies it to the clipboard and briefly shows feedback to the user.
